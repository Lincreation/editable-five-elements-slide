# 30分演習｜AI Sheetsで基本の一巡を試す

## ゴール

配布した架空Excelを使い、AIへ判断基準を渡し、結果を元データと比較し、一箇所だけ修正して、XLSX出力を開き直せます。

合格値：`13件 / 入力858,640円 / 出力858,640円 / 差額0円 / 例外2件（INV-3112、INV-3104-再送）`

使用ファイル：`participant/Genspark研修_飲食店週次支出照合デモ.xlsx`

## 1. 入力を確認する｜5分

- `部門別支出`: 表示順と分類の判断基準
- `請求書明細`: 分類対象。表記揺れ、詳細不足、再送候補を含む

入力前に確認してください。

- [ ] 配布された研修用ファイルである
- [ ] 個人情報・顧客情報・機密が含まれていない
- [ ] 元の請求書IDと金額を変更しないと決めた

## 2. AI Sheetsへ依頼する｜12分

> 部門別支出シートの表示順と判断基準に従い、請求書明細を分類・並べ替えてください。部門コード、部門名、判定理由、確信度、要確認を追加してください。判断材料が不足する行は推測で確定せず要確認とし、確認質問を作ってください。元の請求書IDと金額は変更しないでください。重複候補を検出し、処理前後の総額を照合してください。

## 3. 比較・一箇所修正・XLSX出力・再確認｜8分

- [ ] 部門順が基準シートと一致する
- [ ] 判定理由が読める
- [ ] `INV-3112`を詳細不足の要確認として残し、確認質問がある
- [ ] `INV-3104-再送`を重複候補として残している
- [ ] 元の請求書IDと金額が変わっていない
- [ ] 13件、入力858,640円、出力858,640円、差額0円で一致する
- [ ] XLSXで出力し、開き直して件数・総額・例外2件が残っている

例：

> 要確認の行だけを対象に、確認が必要な理由と担当者への質問を短くしてください。他のセル、順番、金額は変更しないでください。

## 4. 質問・振り返り｜5分

1. AIへ渡した判断基準は何ですか。
2. 人が確認した場所はどこですか。
3. 明日試すなら、Slides・Docs・Sheets・画像のどれですか。

## 操作できない場合

紙上で次の4点を書きます。

- 入力してよい情報
- 欲しい成果物
- AIへ渡す判断基準
- 人が確認して止める場所
