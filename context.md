# Genspark基本研修｜前提

## 仮定

- 対象はAP HoldingsのGenspark利用者。部門横断・初心者を含む約50名を想定する。
- 同一内容を2026-07-22と2026-07-24に実施し、欠席者はアーカイブ視聴と理解度確認で補完する。
- APHD契約プラン、日本語UI、利用可能な出力形式、Team Skills、Connectors、Workflow権限は当日直前に確認する。
- 研修では配布済みの架空データだけを使う。個人情報、顧客情報、機密、本番正本は入力しない。

## コミュニケーションゴール

研修後、受講者がGeminiとGensparkを「正本・権限・保存先・作業場所」で使い分け、AI Slides・AI Docs・AI Sheets・画像生成から適切な入口を選び、SkillsとWorkflowsの違いを説明できる。

## 到達目標

1. GeminiとGensparkを能力の有無ではなく、入力データ・完成物・保存先・権限で選べる。
2. Gensparkの基本操作「頼む→見る→直す→出す」を説明できる。
3. AI Slides、AI Docs、AI Sheets、画像生成の役割と使いどころを説明できる。
4. AIエージェントをLLM、Tools、Context、Loop、Guardrailsで説明できる。
5. Skillは品質、Workflowは順番と自動化を再利用する機能だと説明できる。
6. 配布ExcelをAI Sheetsで処理し、元データ・理由・例外・総額を確認できる。

## 120分

| 区分 | 時間 | 内容 |
| --- | ---: | --- |
| Talk | 60分 | Gemini差分、基本操作、主要4機能、Agent、Skills、Workflows、安全 |
| Show | 30分 | ホーム、Slides、Docs、Sheets、画像、Skills、Workflowsの実画面 |
| Do / Q&A | 30分 | 架空Excelで操作、部分修正、結果確認、質問 |
