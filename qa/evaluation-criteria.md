# 研修資料の評価基準

## 1. 学習設計

- 120分が講義60分、実演30分、参加者操作・質問30分で構成されている
- 到達目標と各章、実演、最後の行動がつながっている
- 参加者が操作し、質問できる余白がある

## 2. 内容の正確性

- Gensparkの機能・Credits・モードは公式Helpに基づく
- Gemini、Excel、Gensparkの比較を誇張しない
- 変更され得るプランや顧客固有条件に「要確認」を付ける
- AIエージェントをLLM、Tool、Context / State / Memory、Environment、Agent loop、Harness / Permissionsで説明できる

## 3. 実務への転用性

- Skillは品質基準、Workflowは手順・分岐として区別できる
- AI Sheets実演に曖昧行、重複候補、総額照合、人の確認が含まれる
- 参加者が自分の業務の判断基準へ置き換えられる

## 4. 安全性

- 架空データのみを使用する
- 外部送信、削除、実データ共有を自動実行しない
- Prompt Injection、最小権限、承認ゲートを説明する
- 最終的な事実、機密、著作権、レイアウトを人が確認する

## 5. 表示・運用品質

- 44ページ、目次44件、講師台本44件の番号とタイトルが一致する
- 1920×1080で文字切れ、重なり、画面外配置がない
- 各ページの見出しだけで主張が分かる
- 出典が講師台本またはスライドに残る
- ローカルで表示し、ブラウザ操作、ページ移動、講師ノート表示を確認する
