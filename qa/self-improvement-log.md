# Genspark基本研修｜5周自己改善ログ

- 実施日: 2026-07-20
- 評価基準: `qa/evaluation-criteria.md`
- 評価体制: 目的・事実、学習設計、デザインの独立3監査＋主担当統合
- 対象: 28枚のスライド、60分講師台本、30分実演、30分操作・質問、参加者資料、実演Excel

## Round 1｜目的から再定義

### 独立評価

- 目的・事実: 39/100
- 学習設計: 46/100
- デザイン: 54/100

### 主な問題

- 「Gensparkが何か」より抽象的なAgent論が前に出ていた。
- Geminiを会話専用のように扱い、両者の実務上の違いが不正確だった。
- 120分のうち、説明・実演・参加者操作の境界が曖昧だった。
- 章扉と似たカード画面が続き、1枚ごとの主張が弱かった。

### 改善

- ゴールを「Geminiとの使い分け」「主要4機能」「Skills / Workflows」「安全な一件の再現」に固定。
- 構成を60分説明・30分画面共有・30分操作と質問に再設計。
- AI Slides、AI Docs、AI Sheets、画像生成・編集を主要機能として前面化し、その他は一覧に縮小。
- 既存構成を継ぎ足さず、28枚へ全面再構成。

## Round 2｜公式情報と技術説明を補正

### 独立評価で判明したこと

- GeminiもGoogle SlidesやSheetsの作成・編集を支援できるため、「Geminiは回答、Gensparkは成果物」という断定は不可。
- Gensparkの現行機能名と、Skills / Workflowsの役割を一次情報で再確認する必要があった。
- AIエージェントは専門語の列挙ではなく、業務上の役割と安全境界まで説明する必要があった。

### 改善

- 比較軸を優劣から「正本・保存先・権限・作業場所」へ変更。
- Genspark Help Center、Google公式ヘルプ、AnthropicのAgent解説を調査ソースに固定。
- Agentを「LLM＝考える」「Tools＝操作する」「Context＝作業メモ」と説明し、観察→判断→実行→確認のLoopを追加。
- 権限、停止条件、人の承認を同じ章に置き、複数モデルでも正しさは自動保証されないと明記。
- WorkflowsのTest RunとTurn Onを区別し、研修ではTest Runまでとした。

## Round 3｜研修として話せる順番へ再設計

### 改善

- 60分説明を、目的5分、比較・Agent17分、基本操作10分、主要4機能16分、Skills / Workflows9分、まとめ3分へ配分。
- 各スライドへ進行目安、説明例、受講者への問いを持つ講師ノートを埋め込み、外部台本と同期。
- 「頼む→見る→直す→出す」と「目的・入力・条件・出力・確認」の2つに基本操作を集約。
- Super Agent、専門機能、Skill、Workflowを4つの役割として整理。
- デザインを暖色の紙面、ネイビー、グリーン、アンバーへ刷新し、比較・工程・図解・作例のレイアウトを使い分けた。

## Round 4｜実演・演習・配布物を同期

### 改善

- 実演を、Home / Super Agent 3分、AI Sheets 12分、AI Slides 5分、AI Docs 3分、画像3分、Skills / Workflows 4分に固定。
- AI Sheetsの確認済みサマリーをコピーし、AI Slides / AI Docsの新しい指示へ貼る順番へ統一。直接連携は前提にしない。
- 参加者操作を、入力5分、依頼12分、比較・修正・XLSX出力・再確認8分、質問5分に統一。
- Excelの正解条件を13件、入力858,640円、出力858,640円、差額0円、例外2件に固定。
- 例外を`INV-3112`（詳細不足）と`INV-3104-再送`（重複候補）で全資料に同期。
- 生成待ちの打切時刻とAI Slides / AI Sheetsのバックアップを実演ガイドへ追加。
- 参加者版をPC・スマホ向けに整え、配布対象をマニフェストで明示。

## Round 5｜独立監査と全範囲回帰

### 実施内容

- 目的・事実、学習設計、デザインの3観点で最終独立監査を実施。
- 別プロセスのCodexレビューで、120分配分、必須論点、実演・操作・質問、代替進行に致命傷なしを確認。
- 28枚を投影モードで目視し、文字切れ、重なり、意図しない改行を修正。
- 目次、前後移動、URL直リンク、編集モード、講師台本パネル、コンソールエラーを回帰確認。
- Excelをスプレッドシート専用ツールで開き、2シート、13明細、858,640円、例外2件、バックアップ差額0円を確認。
- HTML sanitizerと提案資料lintを実行し、エラー・警告0件を確認。

### 最終判定

- 28枚・台本28項目・説明60分が一致。
- 30分実演と30分操作・質問の順序、時間、合格条件が一致。
- 最終独立監査は、目的・事実97/100、学習設計100/100、デザイン96/100。全監査でP0 / P1残件0。
- 別プロセスCodexも「致命傷なし」と判定。
- ローカル研修実施と参加者配布に必要なファイルが揃った。
