# APHD Genspark基本研修

Geminiとの違い、Gensparkの基本操作、AI Slides・AI Docs・AI Sheets・画像生成、Skills・Workflowsを扱う120分研修です。

## 開き方

`index.html`をChrome等で開きます。ローカルファイルのまま動作します。

- `←` `→`: ページ移動
- `F`: 投影表示
- `E`: 画面上の文字・画像を編集
- `S`: 講師ノートを開閉
- `HTML保存`: 編集済みHTMLを保存
- `トーク台本保存`: 講師ノートをMarkdownで保存

## 研修構成

| 区分 | 時間 | 目的 |
